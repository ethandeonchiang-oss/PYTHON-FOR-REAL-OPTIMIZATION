redbulladdiction(fordistrib).py
- This coding project demonstrates the application of lead time, repurchase of inventory at reorder points, as well as the costs associated with purchase habits; all these are adjusted to seasonality.
- Personalized towards my average expenditures on energy drinks (Red Bull), it documents the costs that arise when I buy said energy drink, adjusted to peak seasons such as exams.
- Code holds true with other items, however this code does not account for transaction costs, which could be an added adjustment for future reference.
